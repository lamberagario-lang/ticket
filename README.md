# Эскалада, Totma, X-Caro | Ставрополь

Проект React + Vercel для продажи билетов с генерацией персонализированного PDF-билета.

## Быстрый старт (локально)
1. Установите Node.js (рекомендуется v18+).
2. В папке проекта выполните:
   npm install
   npm run dev
3. Откройте http://localhost:5173

## Деплой на Vercel
1. Залейте проект в репозиторий GitHub.
2. На Vercel: New Project → импортируйте репозиторий.
3. Vercel автоматически определит Vite проект. Deploy.

## Примечания
- После оплаты интеграцию можно настроить так, чтобы платежная система (YooKassa) делала редирект на /thank-you, где можно запускать генерацию билета.
- Версия pdf-lib установлена как ^1.17.1 (совместимая).
